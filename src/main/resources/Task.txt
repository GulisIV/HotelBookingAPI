Тестовое задание по JAVA
Данное задание имеет целью выяснить знания в области web программирования с использованием Java, а также работа с базой данных.
Постановка задачи:
Следует написать небольшое Web приложение, цель которого - бронирование комнат гостиницы.
Приложение должно иметь страницу авторизации (тип авторизации можно выбрать на свое усмотрение), страницу администратора и страницу пользователя. Соответственно в приложении есть два вида пользователей: administrator и user.
Администратор должен иметь возможность:
Создавать комнаты
Устанавливать название, количество спальных мест, цену для комнаты
Задавать время работы для гостиницы(не для комнаты, расписание общее для всех комнат).
Пользователь должен иметь функционал:
Просмотра всех свободных комнат на выбранный промежуток времени
Бронирования свободных комнат с учет времени работы гостиницы
При бронировании уже забронированной комнаты, должна возвращаться ошибка о невозможности бронирования.


Также необходимо написать для приложения JUnit тесты.
- Создание комнаты.
- Создать расписание для работы гостиницы(пн - чт: 09.00 - 21.00, пт: 11.00 - 18.00)
- Для первой комнаты задать перерывы на пн-чт: с 12.00-13.00, пт: 14.00 - 15.00. Для второй комнаты задать перерыв пн-чт: с 11.00-12.00, пт: 13.00 - 14.00.
- Получить все свободные комнаты для понедельника с 09.00 до 12.00. (должна вернуться только первая комната)
- Получить все свободные комнаты с 14.00 до 18.00 для пятницы. (должна вернуться только вторая комната)
- Создать бронирование для первой комнаты с 10.40 - 12.00 (должно создаться бронирование)
- Создать бронирование для первой комнаты с 8.30 - 12.00 (должна вернуться ошибка)
- Создать бронирование для первой комнаты с 12.30 - 14.00 для вторника (должна вернуться ошибка)


 Что использовать(технологии):
Java 8-11(на ваш выбор)
MongoDB
Spring Boot(не ниже 1.5)
Spring Data
Spring Security
RestFul

Фронтенд не обязателен! Достаточно просто API.
